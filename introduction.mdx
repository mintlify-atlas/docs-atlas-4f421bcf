## What is Academia Gema?

Academia Gema is a SaaS platform built for volleyball clubs and training academies. It brings together three distinct user experiences under one roof:

- **Students** can browse the weekly schedule, enroll in classes, report payments, track attendance, and log injuries — all from a mobile-friendly dashboard.
- **Coordinators** (teachers) take attendance during class sessions, manage their profile, and handle the cut-day workflow for billing cycles.
- **Administrators** have full control over every aspect of the academy: locations, courts, training levels, class schedules, student and teacher management, payment validation, publications, benefits, and system settings.

## User roles

Academia Gema uses a role-based access system with three roles:

| Role | Spanish name | Access |
|------|-------------|--------|
| Administrator | Administrador | Full platform access — all management features |
| Student | Alumno | Student dashboard — enrollment, payments, attendance, injuries |
| Coordinator | Coordinador | Teacher dashboard — attendance, profile, cut-day management |

<Note>
  Each role gets a dedicated dashboard layout optimized for their workflow. Unauthorized access attempts are redirected to the `/forbidden` page.
</Note>

## Platform architecture

Academia Gema consists of a React frontend (this application) backed by a REST API and a PostgreSQL database. The frontend is built with:

- **React 19** + **React Router v7** for routing and rendering
- **Tailwind CSS v3** for styling
- **Recharts** for analytics charts
- **Framer Motion** for animations
- **Vite** as the build tool

Explore the [Technical Reference](/api/overview) for full API documentation and [Database Schema](/database/schema) for the data model.
